# my_django_test
***A simple web application that is a knowledge test of the Django framework.***
<hr>

## Main tools:
- ```Python 3.10.2```
- ```Django 4.0.3```
- ```PostgreSQL 14.2```
- ```Pillow 9.1.0```
- ```HTML 5```
- ```CSS 2.1```

## Running applicotion:
- ```python -m pip install requirements.txt```
- ```python .\manage.py runserver```
